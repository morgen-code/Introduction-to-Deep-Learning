# 機械学習の入門的な内容をまとめたノートブックです。

## perceptron

## activetion_function

## xor_model

xor 回路の出力を予測する単純な機械学習モデルを TensorFlow を使って実際に作成し、テストしています。

## janken_model

じゃんけんの勝敗を予測するモデルを作成し、テストしています。

## image_recognition_model
